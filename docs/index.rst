Orange County Lettings
======================

Bienvenue dans la documentation de Orange County Lettings !

Orange County Lettings est une application web Django
 de gestion d'annonces immobilières,
 développée dans le cadre d'un projet de formation.

.. image:: https://img.shields.io/github/languages/top/PierreBulgare/Orange-County-Lettings
    :alt: Langage principal

Présentation du projet
----------------------

Ce projet permet de gérer des profils utilisateurs,
 des locations, des adresses et l'administration associée.

Fonctionnalités principales :

- Gestion des profils utilisateurs
- Gestion des annonces de locations
- Administration des adresses et locations
